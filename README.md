## üá¨üáß Description (English)

Task Tracker CLI is a simple C# console application for managing your task list ‚Äî add, view, delete, and update task statuses.

### Features
- Add tasks with unique id
- View all tasks
- Delete tasks by id
- Mark tasks as done or in progress
- Tasks stored in a plain text file

### How to use
1. Clone the repository:  
   ```sh
   git clone https://github.com/yourusername/task-tracker-cli.git
   ```
2. Open the project in Visual Studio or any C# IDE
3. Build and run `Program.cs`
4. Follow the console menu instructions
5. Tasks are saved in the file `Tasks.cs` in the app directory

| File        | Description                         |
|-------------|------------------------------------|
| Program.cs  | Main logic and console interface   |
| Tasks.cs    | Task class (id, status, description)|
